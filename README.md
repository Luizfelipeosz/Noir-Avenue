## Visão do Produto

A Noir Avenue é uma aplicação web projetada para oferecer uma experiência elegante, intuitiva e segura, construída seguindo princípios de escalabilidade e organização de código.

O desenvolvimento do produto é realizado por sprints, permitindo a implementação gradual de novas funcionalidades e melhorias contínuas.

---

## Funcionalidades Implementadas

* Sistema completo de autenticação de usuários;
* Login de usuários;
* Cadastro de usuários;
* Recuperação de senha;
* Funcionalidade "Lembrar-me";
* Logout do usuário;
* Validação de formulários;
* Sistema de notificações (Toast);
* Estrutura inicial de rotas;
* Identidade visual da plataforma.

## Arquitetura do Projeto

O projeto segue princípios voltados para:

* Componentização;
* Separação de responsabilidades;
* Escalabilidade;
* Organização modular do código;
* Facilidade de manutenção;
* Evolução incremental do produto.

---

## Roadmap

### Sprint 1 - Concluída

* Autenticação de usuários;
* Login e Cadastro;
* Recuperação de senha;
* Toast Notifications;
* Estrutura inicial do Front-End;
* Definição da identidade visual.

### Próximas Sprints

* Dashboard principal;
* Perfil do usuário;
* Persistência de dados;
* Integração com APIs;
* Novas funcionalidades do ecossistema Noir Avenue;
* Otimizações de performance;
* Responsividade avançada.

---

## Objetivos do Projeto

* Desenvolver um produto Front-End escalável;
* Aplicar boas práticas de engenharia de software;
* Simular o ciclo de desenvolvimento de um produto real;
* Evoluir continuamente a arquitetura da aplicação;
* Construir uma base sólida para futuras funcionalidades.
